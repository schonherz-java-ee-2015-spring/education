Blog application: 
Új maven project a homework/{név}/{projectnév} mappában
Blog:
	- id
	- címe
	- szövege
	- (application scope -> minden user ugyanazokat látja)
Megj: Nem kell se file-ba se SQL-be lementeni, viszont application scope miatt amíg a tomcat fut addig benn lesznek a memóriában.
Oldalak:
	- Index: Itt vannak a blogok felsorolva (csak címük)
	- Details: Itt egy blognak az címe és a szövege is (ID alapján ki kell tudni kereseni a blog listából)
	- NewBlog:	meg lehet adni a címet és a szöveget. És miután elposztolta megjelenik az indexen az új is.
		- opcionális: vissza megy az indexre
